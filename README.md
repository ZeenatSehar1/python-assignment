# python-assignment

This repository contains a small Streamlit app `bmi_app.py` that shows a circular BMI gauge using Plotly.

How to deploy
- Push this repository to GitHub (already done).
- Go to https://share.streamlit.io, sign in with GitHub, and click "New app".
- Select the repository `ZeenatSehar1/python-assignment`, branch `main`, and the file `bmi_app.py`.
- Click "Deploy".

Notes
- The repository contains `requirements.txt` with `streamlit` and `plotly` so Streamlit Cloud can install dependencies.
- The app file is `bmi_app.py` at the repository root.
